SEQUENCER
A classic, Buchla inspired 5-stage sequential voltage source. The output of each step can be
adjusted continuously from 0V to 5V with the corresponding potentiometer.

The sequential voltage generates sequences of gates and stored control voltages. It's equipped
with an internal clock, the PULSER that sets the speed of the sequencer, but you can use any other
external clock/trigger source through the EXT. CLOCK input jack to run the sequencer (for example the LFOs or the CV outputs from the touch keyboard).
To create diﬀerent articulations, the GATE switches turns oﬀ the GATE output for each step. This
does not aﬀect the CV voltage output.
The STAGES switch sets the maximum number of steps in the sequence (3, 4, or 5). The PULSER
potentiometer sets the speed of the internal clock generator, this periodic signal is available at the
CLOCK OUT jack.
Touch keyboard also has a built-in sequencer (up to 16 steps). But this sequencer is physically
more convenient to adjust while playing.
To get a Buchla style, use the trigger output of the touch keyboard. Connect this sequencer's
PULSER pulse generator to the keyboard's CLOCK input. Alternatively, connect the keyboard's
internal pulse generator to the sequencer's EXT CLOCK jack.
There are many options for starting the sequencer from external sources.

ROOT NOTE

Sets the root note of the scale applied by the note quantiser from C to H.
The scale is a sequence of notes arranged in ascending order within an octave,
that follows a speciﬁc pattern of intervals (distances between notes). The root
note of a scale is the starting note, the foundational pitch from which the scale is
built and named. The root note anchors the other notes in the scale, giving them
context and deﬁning the key of the music. When you alter the root note, you
eﬀectively transpose the scale to a diﬀerent key, by shifting its entire structure.
This ability to change the root note allows musicians to explore various tonalities
and adapt the scale to ﬁt diﬀerent musical contexts.
The adjustable root note is especially useful when playing a synthesizer in a
precise musical context, where staying in tune with other instruments is critical.
In these situations, setting the root note ensures that the oscillators (VCOs) are
tuned correctly relative to the rest of the ensemble, maintaining harmony and
consistency across instruments.
However, if the synthesizer is being used on its own, without the need to align
with other tuned instruments, the root note parameter becomes less essential. In
this case, you can achieve the same eﬀect by manually tuning
the VCOs to transpose the base pitch along with the entire scale. This manual
adjustment eﬀectively shifts the scale’s pitch center, making the root note parameter redundant for solo performances or experimental setups where strict tuning
isn’t necessarily required.
Adjusting the scale and root note can lead to duplicate notes on the controller
keyboard because both parameters aﬀect the output of the quantizer. When the
scale or root note is shifted, the quantizer remaps incoming notes to ﬁt the new
settings, which can cause certain notes on the keyboard to produce the same
pitch, resulting in duplicates. After making adjustments to the scale and root note,
users may need to ﬁne-tune the pre-programmed notes on the keyboard controller to eliminate these duplicates. By reprogramming or reassigning notes, users
can ensure that each key corresponds uniquely to a pitch within the new scale.

CLOCK

Sets the tempo of the keyboards internal clock within a range of 10 to 300 BPM
(Beats Per Minute). While this range might seem limited at ﬁrst, it can be signiﬁcantly extended by utilizing the arpeggiator’s clock multiplication/ division parameter.
When a clock pulse is received through the CLOCK INPUT jack, the keyboard will
automatically switch to using the external clock source for synchronization. To
revert back to the internal clock, all you need to do is modify the BPM setting. By
adjusting the tempo value manually, the keyboard will resume using its internal
clock for tempo generation.

NUANCES AND PECULIARITIES
You may still ﬁnd some nuances. Please let us know!
SOLAR 42F is a fully analogue instrument, except for the touch-sensitive multifunctional
keyboard and eﬀector which are semi-digital. All oscillators, drones, mixer, preamp and
ﬁlter are analogue.
Since SOLAR 42F has two eﬀector circuits, some eﬀects on the left and right channels may
diﬀer slightly from each other, creating a "live" stereo sound.
When GAIN is maxed out in the Preamp you'll hear a touch keyboard operation noise.
You will hear a click when you switch the LP/BP button in the ﬁlter unit. This is normal.
Do not use the maximum position of the MASTER knob or headphones. You will hear
eﬀector noise. Use the maximum volume on the mixer. And use the MASTER or headphone
volume control at the maximum 3 o'clock position. Since there is no VCA on the eﬀector
output.

low/band switches in the ﬁlter;

LED indication in the joystick unit;
new graphics, output connectors are marked in red colour, input connectors in black
colour;
colour coding of potentiometer knobs by blocks;
sub generators in voices with volt bounce;
most toggle switches in the blocks have been replaced by more convenient push-button switches;
VCO A / VCO B voices dry outputs moved to the front panel;
internal noise is minimised as much as possible.
